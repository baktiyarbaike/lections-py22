CTRL+ALT+T - КОМАНДА ОТКРЫТИЯ ТЕРМИНАЛА
~ - СИМВОЛ ДОМАШНЕЙ ДИРЕКТОРИИ (ПАПКИ)
/ -  символ корневой директории (папки)
.. - пердыдущую директорию
. - текущую директорию 
1.ls (list) -> выводит список всех директорий и файлов в текущей директории (рабочая директория) 
ls -a -> выводит список как и скрытых так и обычных файлов и директории
ls -l -> выводит список директории и файлов с подробной инфой

2. cd (cange directory) -> меняет текущую директорию на указанную
cd <название директории>

CTRL+SHIFT+C - копировать в терминале
CTRL+SHIFT+V - вставить в терминале

3.mkdir (make directory) -> создает указанную директорию
mkdir <директория>
4. touch -> создает указанный файл
touch <name_of_file>.<extension>(расширение (txt, docs))>
5.pwd (print working directory) -> выводит путь по текущей директории 
6.rmdir (remove directory) -> удаляет указанную директорию(не может удалить не пустую директорию)
7. rm -> удаляет файл
rm <name_of_file>
rm -rf -> принудительное удаление файла либо директории

8. mv (move) -> перемещает либо переименовывает файл либо директорию, в зависимости от указанного второго аргумента
mv <файл_директория> <новое_имя> -- директория/>

9.cat -> выводит содержимое указанаго файла

10. nano -> открывает графический интефейс в терминале для редактирования файла
*если нет указаного файла , то создадится новый с таким именем 
nano <name_of_file>

11. echo -> команда для быстрой записи данных в файл через терминал

echo текст для записи > название файла

12. cp -> команда для копирования файла в нужную директорию
cp <file_to_copy> <path>
